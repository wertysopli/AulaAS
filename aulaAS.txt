Dentro do git
PWD = indica o diretorio do arquivo
clear = limpa a tela
ls = mostra o que contem dentro da pasta
ls -a = mostra os arquivos ocultos 
ls -l = exibe as informações mais detalhadas da pasta         
ls -h = (h de human) mostra as informações mais facil de entender
cd .. = volta uma pasta 
cd "nomedapasta" = abre/muda de pasta
ls -lah = junção dos ls (informação ainda mais detalhada)
d = nada / rwx / r = read(permissão de leitura) / write = escrita (editavel) /
x= execute
. = pasta atual
.. = pasta anterior
